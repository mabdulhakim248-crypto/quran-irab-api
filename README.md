# 🕌 API إعراب القرآن الكريم

واجهة برمجية مجانية وشاملة للبحث في إعراب وتحليل كلمات القرآن الكريم.

## 📚 المصادر
- **إعراب الكربلائي** — إعراب تفصيلي لجميع كلمات القرآن (66,516 كلمة)
- **MASAQ Dataset** — تحليل صرفي ونحوي علمي (157,676 مدخل)

## 🚀 التشغيل المحلي

```bash
# 1. تثبيت المتطلبات
pip install -r requirements.txt

# 2. تحميل قواعد البيانات
python download_data.py

# 3. تشغيل الـ API
python main.py
```

## 🔍 نقاط النهاية

| المسار | الوصف |
|--------|-------|
| `GET /api/irab/search?q=كلمة` | البحث في إعراب الكربلائي |
| `GET /api/irab/ayah/{سورة}/{آية}` | إعراب آية كاملة |
| `GET /api/irab/surah/{سورة}` | إعراب سورة كاملة |
| `GET /api/morphology/search?q=كلمة` | البحث في MASAQ الصرفي |
| `GET /api/morphology/ayah/{سورة}/{آية}` | التحليل الصرفي لآية |
| `GET /api/morphology/word/{سورة}/{آية}/{كلمة}` | تحليل كلمة بعينها |
| `GET /api/stats` | إحصاءات قاعدة البيانات |
| `GET /docs` | توثيق تفاعلي (Swagger UI) |

## 💡 أمثلة

```
GET /api/irab/ayah/1/1        → إعراب البسملة
GET /api/irab/ayah/2/255      → إعراب آية الكرسي
GET /api/irab/search?q=مجرور  → البحث عن كلمة مجرور
GET /api/morphology/ayah/36/1 → تحليل يس
```
